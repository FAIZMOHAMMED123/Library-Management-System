# Library Management System

A robust console-based application implemented in **Python** using solid **Object-Oriented Programming (OOP)** principles.

## 🚀 Key Features & OOP Concepts

* **Inheritance:** Implemented cleanly across classes using `super().__init__()` for common attributes in `Book` and `Magazine`.
* **Encapsulation:** All data properties are strictly private (`__attributes`) and securely accessed or modified only via explicit public getters and setters.
* **Polymorphism:** Standardized information formatting by overriding the `display_info()` method across different object types.
* **Core Business Logic:** Manages complex relations securely (checking availability, preventing duplicate borrows, and handling error checking for missing IDs during transactions).
